# CSS Theme Switcher

## Building and running on localhost

First install dependencies:

```sh
npm install
# or
yarn install
```

To run:

```sh
npm run start
# or
yarn start
```

To create a production build:

```sh
npm run build-prod
# or
yarn build-prod
```

To create a development build:

```sh
npm run build-dev
# or
yarn build-dev
```

## Running

```sh
node dist/bundle.js
```
